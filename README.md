## Included Texts

- **Ramcharitmanas**
- **Valmiki Ramayana**
- **Rigveda**
- **Atharva Veda**
- **Yajurveda** (Kanva & Madhyandina Samhitas)
- **Srimad Bhagavad Gita**
- **Mahabharata** (Critical Edition)

The texts are parsed and structured into a common `Verse` format with metadata, verse identifiers, and optional translations or commentaries.
